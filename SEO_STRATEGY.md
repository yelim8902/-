# 벌칙 룰렛 SEO & 바이럴 마케팅 전략

## 🎯 목표
- 구글 검색 상위 노출 (키워드: "벌칙 룰렛", "커플 룰렛", "19금 룰렛" 등)
- 소셜 미디어 바이럴 확산
- 장기적으로 광고 수익 모델 구축

---

## 📋 1단계: SEO 기본 최적화 (즉시 적용 가능)

### ✅ 완료된 항목
- [x] 메타 태그 (title, description, keywords)
- [x] Open Graph 태그 (소셜 공유 최적화)
- [x] Twitter Card 태그
- [x] 구조화된 데이터 (JSON-LD)
- [x] robots.txt
- [x] sitemap.xml

### 🔧 추가 작업 필요
1. **실제 도메인 URL로 업데이트**
   - `your-domain.vercel.app` → 실제 Vercel 도메인으로 변경
   - 또는 커스텀 도메인 사용 (예: `penalty-roulette.com`)

2. **OG 이미지 생성**
   - 크기: 1200x630px
   - 내용: 룰렛 이미지 + "벌칙 룰렛" 타이틀
   - 파일명: `og-image.png` (public 폴더에 저장)

3. **Favicon 추가**
   - 32x32px, 16x16px 아이콘 생성
   - 파일명: `favicon.png`

## 📊 2단계: Google 서비스 연동

### Google Search Console 설정
1. https://search.google.com/search-console 접속
2. 속성 추가 → URL 접두어 입력 (실제 도메인)
3. 소유권 확인 (HTML 태그 또는 파일 업로드)
4. sitemap.xml 제출: `https://your-domain.com/sitemap.xml`

### Google Analytics 설정
1. https://analytics.google.com 접속
2. 측정 ID 발급 (예: G-XXXXXXXXXX)
3. `index.html`에 추적 코드 추가

### Google Tag Manager (선택사항)
- 더 세밀한 이벤트 추적이 필요할 때 사용

---

## 🚀 3단계: 성능 최적화

### Core Web Vitals 개선
- **LCP (Largest Contentful Paint)**: < 2.5초
- **FID (First Input Delay)**: < 100ms
- **CLS (Cumulative Layout Shift)**: < 0.1

### 최적화 방법
1. **폰트 최적화**
   - 현재: Google Fonts 사용 중 ✅
   - 추가: `font-display: swap` 적용

2. **이미지 최적화**
   - OG 이미지: WebP 형식 사용
   - 압축: TinyPNG 등으로 최적화

3. **코드 최적화**
   - CSS/JS minify (배포 시 자동)
   - 불필요한 코드 제거

## 🔥 6단계: 바이럴 마케팅 전략

### 소셜 미디어 전략

#### 1. 인스타그램/틱톡
- **콘텐츠 아이디어**:
  - "룰렛 돌린 결과" 스크린샷/영상
  - "가장 수위 높은 벌칙" 랭킹
  - "커플들이 룰렛 돌린 후기"
- **해시태그**: #벌칙룰렛 #커플게임 #19금 #데이트아이디어

#### 2. 카카오톡 오픈채팅방
- 관련 오픈채팅방에 링크 공유
- "친구들과 보드게임할 때 유용한 사이트"로 소개

#### 3. 커뮤니티 마케팅
- **네이버 카페**: 보드게임, 커플 관련 카페
- **디시인사이드**: 게임 갤러리
- **레딧**: r/korea 등

#### 4. 인플루언서 협업
- 소규모 인플루언서에게 무료 제공
- 리뷰/사용 후기 영상 제작 요청

### 바이럴 요소
1. **공유 인센티브**
   - "친구 3명 초대하면 특별 벌칙 해제" (선택사항)

2. **재미있는 기능**
   - 랜덤 벌칙 생성기
   - 벌칙 난이도 조절
   - 결과 공유 기능

3. **트렌드 활용**
   - 계절별/이벤트별 특별 벌칙 추가
   - 밸런타인데이, 화이트데이 등
